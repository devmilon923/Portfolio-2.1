# 🌐 My Portfolio Website

Hey! 👋  
This is my **official portfolio website**.

The primary goal of this portfolio is to showcase my personal brand through a clean, modern, and carefully crafted user experience. Instead of adding unnecessary features, I focused on creating an aesthetic design that reflects my design taste and development approach.

## ✨ Design Philosophy

- 🌙 Dark theme throughout the entire website.
- 🪨 Built using a **Claymorphism** design style.
- 🎨 Every component is carefully designed to maintain a consistent visual language.
- 📐 Strong emphasis on spacing, typography, animations, and overall user experience.
- 💙 This is the portfolio I'm genuinely happy with and represents my current design standard.

## 💻 Development

This project was primarily developed by **me**.

AI assisted with approximately **10–15%** of the development process, mainly for:

- Code polishing
- Improving code structure
- Adding helpful comments
- Minor optimizations and refactoring

The overall architecture, implementation, design decisions, and development were done by me.

## 🚀 Deployment

This portfolio is deployed **for free using GitHub Pages**.

Since this is a **Next.js** project, GitHub Pages doesn't support it out of the box. I spent time researching and configuring a **GitHub Actions** workflow that automatically builds and deploys the project whenever changes are pushed.

This setup allows me to:

- Deploy the site automatically
- Host it completely free on GitHub Pages
- Use my own custom domain

## ❤️ Final Note

This project is more than just a portfolio—it's a reflection of my growth as a developer and designer. Every detail, from the UI to the deployment pipeline, was built with care.

I hope you enjoy exploring it as much as I enjoyed building it.
